# EcoRoute — Eko Yo'l 🌿

**EcoRoute** — O'zbekiston universitetlari uchun yaratilgan, interaktiv xarita asosidagi transport CO₂ emissiyasi kalkulyatori.

# Havola ishlamoqda
https://ekoyol.netlify.app/

## Loyiha haqida

Foydalanuvchi ikki manzilni (boshlang'ich va borish joyi) tanlaydi, dastur esa turli transport vositalari — **avtomobil, avtobus, mototsikl va velosiped** uchun yo'nalishni va har birining CO₂ chiqindilarini hisoblab beradi.

## Asosiy imkoniyatlar

- 🗺️ Interaktiv xarita (faqat O'zbekiston hududi)
- 🔍 Manzil qidirish va avtoto'ldirish (LocationIQ API)
- 📊 Transport turlariga ko'ra CO₂ emissiyasini solishtirish
- 💡 Yillik kampus emissiyasidagi ulushni ko'rsatish
- 🚴 Velosiped yoki avtobus tanlash orqali tejaladigan emissiya

## Texnologiyalar

| Texnologiya | Maqsad |
|---|---|
| HTML / CSS / JavaScript | Asosiy front-end |
| Bootstrap 5 | UI va responsive dizayn |
| Leaflet.js | Interaktiv xarita |
| LocationIQ API | Yo'nalish va manzil qidirish |

## Ishga tushirish

Loyihani ishga tushirish uchun `index.html` faylini brauzerda oching — hech qanday o'rnatish talab etilmaydi.
